# Clash-of-Clans
项目介绍

 本项目是程序设计范式课程的实践作业，参考经典游戏“部落冲突”的核心玩法，融合资源采集、建筑发展、兵种训练、战斗模拟四大核心模块，打造轻量化即时战略+经营类游戏。项目基于 Cocos2d 游戏引擎开发，采用 C++ 作为主要编程语言，聚焦于游戏核心框架构建与逻辑实现，不涉及商业盈利目的。

基础功能：

 1.主村庄 / 大本营系统

 2.三类资源：

 金币 圣水 人口/容量

 3.支持不同类型建筑：

 资源生成建筑（矿场/水收集器） 军营（生成士兵） 防御塔（弓箭塔/加农炮）

 存储建筑

 4.支持常见兵种：

 弓箭手 野蛮人 炸弹人 巨人（高血量肉盾）

 5.AI 战斗系统：

 路径寻找 攻击判定 建筑优先级（如优先打防御塔） 多建筑等级：至少 3 级

 6.一局战斗流程：

 投放兵种 兵种自动战斗 胜负判定

 7.支持至少 2 张不同地图

 8.背景音乐 & 战斗音效

技术栈

 游戏引擎：Cocos2d-x（v3.17）

 开发语言：C++（C++11 及以上标准）

 开发工具：Visual Studio（Windows）/ Xcode（macOS）、CMake
